# KD Autocars - Premium Cars for Rent & Sale

A modern, premium car rental and sales website built with React, TypeScript, and Tailwind CSS.

## Features

- Premium car listings for rent and sale
- Albanian language support
- Modern, responsive design
- Dark theme with yellow/gold accents
- Smooth animations and transitions
- WhatsApp integration for instant contact
- Instagram integration (@kd.autocars)

## Tech Stack

- **Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui
- **Routing**: React Router
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd kd-autocars-elevate-drive

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Development

The project uses:
- Vite for fast development and building
- TypeScript for type safety
- Tailwind CSS for styling
- React Router for navigation

## Building for Production

```sh
npm run build
```
